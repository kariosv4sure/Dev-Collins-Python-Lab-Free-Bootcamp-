Minimalist, dark-mode landing page for a free Python bootcamp. 10 slots only. No frameworks. No backend. Pure HTML/CSS/JS.

Features

· Persistent slots — 10 minus stored emails, survives refresh
· Waitlist mode — auto-activates when slots = 0
· LocalStorage email list — no "clear all" button, individual delete only
· Countdown timer — to next session (6:30PM WAT daily)
· Telegram rule — must DM @Just_Collins101 after sign-up
· Fully responsive + dark mode
